# 目前篩選後建議採用的問卷組合

## 主組合

| 研究面向 | 主問卷 | 補強問卷 | 理由 |
|---|---|---|---|
| 自我效能 | PSEQ 或 MSLQ self-efficacy | SEMLI-S、SMQ-II、SOSESC-P | PSEQ/SOSESC-P 可把研究穩定落在物理；MSLQ/SEMLI-S 可接 SRL |
| 自我評量/自我評估 | SEMLI-S monitoring/evaluation/planning | MSLQ metacognitive self-regulation | SEMLI-S 最貼近科學學習，MSLQ 支撐 SRL 理論主架構 |
| 自我監控 | MSLQ metacognitive self-regulation | SEMLI-S monitoring/evaluation/planning | 可測學習中檢查理解、覺察困難與調整策略 |
| 目標設定與計畫 | MSLQ goal orientation | SEMLI-S monitoring/evaluation/planning | 可改寫成物理學習目標與計畫 |
| 反思與調整 | MSLQ effort regulation / metacognitive self-regulation | SEMLI-S monitoring/evaluation/planning | 對應策略修正、努力維持、學習後調整 |
